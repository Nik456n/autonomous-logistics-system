
# autonomous-logistics-system
Автономная логистическая система для управления флотом роботов на производственном цехе (конструктор , сервер , визуализатор).
# Система автономной логистики роботов

Симулятор управления флотом автономных роботов-перевозчиков на производственном цехе.  
Разработан в рамках международного фестиваля научно-технического творчества (капитан команды и ведущий разработчик).  
**Результат:** 5 место в финале фестиваля.

# Архитектура

Система состоит из трёх компонентов:

- **constructor.html** — графический редактор планировки цеха (стабильность 90-95%)
- **server.py** — Flask-сервер с интеграцией GigaChat AI (рабочий)
- **visualizer.html** — визуализатор симуляции в реальном времени (в разработке, основной функционал работает, система приоритетов и стабильность требуют доработки)

# Как работает

1. В конструкторе пользователь рисует цех: склад, точки погрузки/выгрузки, зарядные станции, препятствия, тоннели, задаёт типы роботов.
2. Данные отправляются на сервер (`POST /optimize-logistics`).
3. Сервер генерирует контроллер через GigaChat API (или использует встроенный алгоритм).
4. Визуализатор запускает симуляцию в реальном времени (A*, приоритеты, резервирование).

# Технические особенности

- **A* pathfinding** с диагональным движением и динамической сеткой препятствий
- Управление зарядом батареи — автоматическая отправка на зарядку при <30%
- Балансировка нагрузки между роботами
- Соответствие типов грузов между точками погрузки и выгрузки
- Поддержка тоннелей как оптимизированных маршрутов
- Интеграция с **GigaChat AI** для генерации логики управления
- Fallback-режим — работает без AI на встроенном алгоритме

# Запуск

# Требования
bash
pip install flask flask-cors requests

1. Настройка GigaChat (опционально)

В server.py укажите свои учётные данные:
python
CLIENT_ID = "ваш_client_id"
CLIENT_SECRET = "ваш_client_secret"
GIGACHAT_ENABLED = True

Получить ключи: https://developers.sber.ru/gigachat

# 2. Запуск сервера
bash
python server.py

Сервер запустится на http://127.0.0.1:8030

# 3. Открыть конструктор

Открыть constructor.html в браузере, спроектировать цех и нажать «Оптимизировать» — визуализатор откроется автоматически.
